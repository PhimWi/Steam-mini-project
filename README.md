# Steam-mini-project

Steam Games Dataset เป็นข้อมูลที่เกี่ยวข้องกับเกมในแพลตฟอร์ม Steam ซึ่งเป็นแพลตฟอร์มที่ได้รับความนิยมในการรวบรวมและจัดจำหน่ายเกมดิจิทัล การวิเคราะห์ข้อมูลนี้มีวัตถุประสงค์เพื่อทำความเข้าใจแนวโน้มและปัจจัยต่าง ๆ ที่ส่งผลต่อยอดขายและความนิยมของเกม ซึ่งอาจช่วยให้ผู้พัฒนาเกม นักการตลาด หรือผู้สนใจในวงการเกมสามารถวางกลยุทธ์และตัดสินใจได้ดีขึ้นในการพัฒนาเกมในอนาคต

Dataset imported from: https://www.kaggle.com/datasets/fronkongames/steam-games-dataset
